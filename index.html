Pesquisa Sobre Frameworks de desenvolvimento

  1. React

    O React trata se de uma ferramente com o objetivo de auxiliar no desenvolvimentode interfaces para o usuário baseando se em si em uma biblioteca JavaScript.Surgiu em 2011 no uso de interfaces no Facebook e partir disso acabou sendo usada por muitos desenvolvedores em suas aplicações como o Instagram por exemplo

  2. Materialize

    O Materialize trata se de um framework voltado para o uso no ambiente de desenvolviment front end sendo muito semelhante em alguns aspectos ao bootstrap mas com um diferencial de ser mais leve nas nas suas alterações na interface quando aplicado e com isso a interface fica com um aspecto mais agradavél e reponsivo um fator extremamente importante dos dias de hoje

  3. Angular

    O Angular é um framework usado para fazer interfaces em aplicações com a utilização de HTML,CSS e JS ,foi criado pelo Google e sua capacidade de criar páginas com qualidade é admirável
